# Idle Ant Colony - Support

Welcome to the official support repository for **Idle Ant Colony**!

## About the Game

Idle Ant Colony is an idle/incremental game where you build and manage your own ant colony. Tap to collect honeydew, recruit different ant types, unlock upgrades, and prestige to grow your empire!

## How to Get Help

### Report a Bug
Found something not working correctly? [Open a bug report](../../issues/new?template=bug_report.md)

### Request a Feature
Have an idea to make the game better? [Submit a feature request](../../issues/new?template=feature_request.md)

### General Questions
For general questions or feedback, [start a discussion](../../issues/new?template=question.md)

## Before Submitting

1. **Search existing issues** - Your question may already be answered
2. **Include device info** - iPhone model and iOS version help us debug
3. **Be descriptive** - The more detail, the faster we can help
